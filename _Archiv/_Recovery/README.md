# _Recovery

Dieses Verzeichnis ist ein Archiv- und Referenzbereich.

## Zweck
- PDB-Dokumentlisten
- Recovery-Metadaten
- Listen fehlender Dateien und Hotfile-Hinweise
- Rohmaterial für den kontrollierten Wiederaufbau

## Wichtiger Hinweis
- Inhalte hier sind nicht die produktive Quellbasis.
- Änderungen im aktiven Code sollen nicht so dokumentiert werden, als kämen sie direkt aus diesem Verzeichnis.
- Wenn Recovery-Material herangezogen wird, sollte die aktive Implementierung danach in den regulären Projekten und der regulären Doku nachvollziehbar bleiben.
